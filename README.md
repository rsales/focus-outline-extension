# Focus Outline Helper

A simple Chrome extension that highlights focused elements with a red dashed outline to improve accessibility and keyboard navigation.

## ✨ Features

- Toggle per website
- Visual focus indicator
- Dynamic icon + badge
- No page reload required

## 🚀 Installation (Development)

1. Clone the repository:

```bash
git clone https://github.com/rsales/focus-outline-extension.git
```
2. Enable **Developer Mode**
3. Click **Load unpacked**
4. Select the project folder

## 📦 Usage

- Click the extension icon
- Enable/disable per site
- Badge "ON" indicates active state

## 🛠 Tech

- Chrome Extensions Manifest V3
- Vanilla JavaScript
